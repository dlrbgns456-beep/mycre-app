❓ 자주 묻는 질문 (FAQ)

로그인이 안 돼요

볏마루도감 iOS 앱에서는 이메일/비밀번호 로그인만 지원합니다.

  • 회원가입 화면에서 이메일 인증 메일을 확인했는지 확인해 주세요.
  • 비밀번호 분실 시 로그인 화면 하단의 "비밀번호 잊으셨나요?"를 통해 재설정할 수 있습니다.
  • 인증 메일이 안 오면 스팸 메일함을 확인해 주세요.
  • 그래도 안 되면 위 이메일로 가입 이메일 + 발생 상황을 보내주세요.
계정/데이터를 삭제하고 싶어요

앱 내에서 직접 삭제 가능합니다:

  • 앱 실행 → 하단 [더보기] 탭 → [회원 탈퇴]
  • 탈퇴 시 등록한 모든 개체 정보·사육 기록·게시물이 즉시 삭제됩니다 (복구 불가).
  • 분양/입양 거래 추적이 필요한 게시물은 작성자 정보를 익명화한 채 보존될 수 있습니다.
  • 앱 접근이 어려우면 위 이메일로 가입 이메일과 함께 삭제 요청을 보내주세요.
AI 모프 판별 무료 횟수가 다 떨어졌어요

매일 3회 무료 판별이 제공되며, 한국 시간 자정(KST 00:00)에 자동 리셋됩니다.

  • 오늘의 무료 횟수를 소진하면 다음 날 자정에 다시 사용할 수 있어요.
  • 판별 결과는 참고용이며, 정확한 판별을 위해서는 전문 브리더의 확인을 권장합니다.
커뮤니티에서 부적절한 게시물/댓글을 봤어요

각 게시글/댓글의 신고 버튼을 통해 즉시 신고할 수 있습니다.

  • 운영자는 신고 접수 후 24시간 이내 콘텐츠 검토 → 위반 시 즉시 삭제 + 작성자 제재합니다.
  • 특정 사용자의 모든 글/댓글을 안 보이게 하려면 차단 기능을 이용해 주세요. 차단한 사용자의 콘텐츠는 즉시 피드에서 사라집니다.
  • 긴급한 경우 위 이메일로 신고 내용을 직접 보내주셔도 됩니다.
차단한 사용자를 다시 보이게 하려면?

[더보기] 탭 → [차단 관리] 메뉴에서 차단 목록을 확인하고 해제할 수 있습니다.

분양/입양 거래 관련 문의

볏마루도감은 거래 중개 플랫폼이 아니며, 직거래 또는 외부 안전결제(네이버페이 안전결제 등)를 권장합니다.

  • 선입금은 절대 권장하지 않습니다.
  • 거래 분쟁이 발생한 경우 위 이메일로 거래 내역과 함께 신고해 주세요.
  • 거래 관련 보존 정보는 개인정보처리방침에 따라 처리됩니다.
데이터가 동기화 안 돼요 / 앱이 느려요

먼저 시도해 보세요:

  • 인터넷 연결 확인 (Wi-Fi ↔ 데이터 전환)
  • 앱 완전 종료 후 재실행
  • 최신 버전인지 확인 (App Store에서 업데이트 확인)
  • 그래도 안 되면 위 이메일로 사용 기기 모델 + iOS 버전 + 발생 시점을 알려주세요.
